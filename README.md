# dunefetch

neofetch + falling sand engine for your terminal 

## install 

```
git clone https://github.com/datavorous/dunefetch
cd dunefetch
python -m venv .venv
source .venv/bin/activate
dunefetch
```

## help

```
dunefetch --help
dunefetch --help-controls
dunefetch --version
```
